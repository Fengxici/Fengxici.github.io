由于spring cloud微服务架构都是基于spring boot构建的，所以我们就可以是很方便使用一种统一的方式来监控这些服务。Spring Boot Admin正式这样一个用于管理和监控一个或多个Spring Boot程序的组件。其在 Spring Boot Actuator 的基础上提供了简洁的可视化 WEB UI，提供如下功能：

显示 name/id 和版本号
显示在线状态
Logging 日志级别管理
JMX beans 管理
Threads 会话和线程管理
Trace 应用请求跟踪
应用运行参数信息，如：

Java 系统属性
Java 环境变量属性
内存信息
Spring 环境属性
